# Cities Weather Dashboard Project

## Overview

This project collects **current weather and climate data** for major cities, stores it in a **SQLite database**, and allows users to **query the data** using a simple command-line interface. The project also includes a **Python scraper** to gather data from the [Time and Date Weather site](https://www.timeanddate.com/weather/). Additionally, the project uses **data visualizations with graphs** to analyze and explore weather patterns across cities.

### Project Structure

```
project_root/
README.md
dashboard/
app.py # Display data visualization graphs
web scraping/
scraper.py # Scrapes weather and climate data
query_db.py # Command-line database query tool
database/
cities_weather.db # SQLite database
data/
raw/ # CSV files generated by scraper
```

## Features

- Scrape current weather and climate data for multiple cities.
- Store data in a **SQLite database** (`weather_data` and `climate_data` tables).
- Query the database via a **command-line interface**.
- View joined weather and climate data in a simple, readable table.
- **Data visualizations** with graphs are used to analyze weather patterns and compare city climates.
- Designed for beginners to learn **web scraping, SQLite, and SQL joins**.

## Getting Started

### 1. Install Dependencies

Make sure you have Python 3 installed. Install dependencies and libraries in the requirements.txt file.

## 2. Run the Scraper

From the project root:
python "web scraping/scraper.py"
CSV files will be saved in data/raw/.
Database tables will be created in database/cities_weather.db.

## 3. Query the Database

From the project root:
python "web scraping/query_db.py"
The program will list tables in the database.
It will run a simple join query to display current weather and climate metrics per city.
You can type your own SQL queries at the prompt.
Type exit to quit the CLI.

## 4. Example Query

Inside the CLI:

```
SELECT
weather_data.city,
weather_data.current_temp,
weather_data.humidity,
weather_data.wind,
weather_data.forecast,
climate_data.metric,
climate_data.value,
climate_data.month
FROM weather_data
JOIN climate_data ON weather_data.city = climate_data.city
LIMIT 20;
```

This will show current weather for each city along with climate metrics.

## 5. Run Streamlit visualization

From the project root:
streamlit run dashboard/app.py
